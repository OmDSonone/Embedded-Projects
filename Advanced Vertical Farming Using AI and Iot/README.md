# AI-Powered Smart Agriculture System

## Project Overview

The AI-Powered Smart Agriculture System is an IoT and Artificial Intelligence based solution designed for smart farming and vertical farming applications. The system collects environmental data using ESP32 sensors, captures plant images through an ESP32-CAM module, detects plant diseases using AI models, and displays real-time information through a web dashboard.

## Components

### 1. ESP32 Sensor Module

Collects environmental parameters such as temperature, humidity, soil moisture, and other sensor readings.

### 2. ESP32-CAM Module

Captures images of plants and sends them for disease detection and monitoring.

### 3. AI Model

Uses Machine Learning and Deep Learning techniques to:

* Detect plant diseases from images
* Analyze crop health
* Support smart farming decisions

### 4. Web Dashboard

A web-based dashboard that displays:

* Sensor readings
* Disease detection results
* Environmental monitoring data
* System status and analytics
