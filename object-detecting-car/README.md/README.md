\# Object Detecting Car (Obstacle Avoidance Robot)



\## Description

This project is an Arduino-based object detecting car designed to avoid

obstacles automatically. The robot uses an ultrasonic sensor to detect

objects in front of it and changes its direction to prevent collision.



The project demonstrates basic embedded systems concepts such as sensor

interfacing, motor control, and real-time decision making.



\## Working Principle

1\. Ultrasonic sensor continuously measures distance

2\. If an obstacle is detected within a threshold distance:

   - The car stops

   - Changes direction automatically

3\. If no obstacle is detected:

   - The car moves forward



\## Components Used

\- Arduino UNO

\- Ultrasonic Sensor (HC-SR04)

\- Motor Driver (L298N / L293D)

\- DC Motors

\- Robot Chassis

\- Power Supply

\- Jumper Wires



\## Features

\- Automatic obstacle detection
