\# IoT-Based Smart Parking System with Automated Entry \& Exit Control



\## Description

This project is an IoT-based smart parking system that allows users to

select a parking slot from a website. Once a slot is selected, the entry

gate opens automatically. At the time of exit, the user must complete

payment first, after which the exit gate opens.



The system ensures efficient parking management using real-time control

and automation.



\## System Workflow

1\. User opens the smart parking website

2\. Available parking slots are displayed

3\. User selects a vacant slot

4\. Entry gate opens automatically

5\. Parking duration is calculated

6\. User pays parking charges via UPI

7\. Exit gate opens only after successful payment



\## Hardware Used

\- NodeMCU ESP8266

\- Arduino UNO

\- Servo motors (Entry \& Exit gates)

\- Power supply

\- Jumper wires



\## Software \& Technologies

\- HTML, CSS, JavaScript (Frontend)

\- Arduino IDE / Embedded C

\- ESP8266 Wi-Fi communication



\## Communication

\- Website ↔ NodeMCU (Wi-Fi based direct communication)

\- NodeMCU ↔ Arduino UNO (Serial communication)



\## Features

\- Online parking slot selection

\- Real-time slot status display

\- Automatic entry gate control

\- Payment-based exit control

\- Low-cost and scalable solution
